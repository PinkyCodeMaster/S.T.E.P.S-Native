## 1. Purpose

Defines how S.T.E.P.S supports users when something goes wrong. Recovery beats optimisation, engagement, or performance metrics. If a conflict arises, recovery wins.

## 2. Core Principles

3. **Choice, Not Command:** Offer options, never orders.
4. **Containment Over Perfection:** Stop the bleed; perfection can wait.
5. **Recovery Is Progress:** Using recovery tools is a positive action.
